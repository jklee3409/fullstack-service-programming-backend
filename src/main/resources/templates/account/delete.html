GitInsight 계정 및 데이터 삭제 요청

이 페이지는 Google Play 데이터 보안 정책에 따라, 사용자가 GitInsight에서 계정 및 관련 데이터를 삭제 요청하는 방법을 안내하기 위해 제공됩니다.

1. 계정 및 데이터 개요

GitInsight는 GitHub OAuth를 사용하여 사용자를 인증합니다.

앱은 다음과 같은 정보를 서버에 저장할 수 있습니다.

  • GitHub 사용자 ID 및 공개 프로필 정보
  • 저장소 및 커밋 메타데이터 (예: 저장소 이름, 커밋 메시지 요약)
  • 앱 사용과 관련된 로그 및 통계 데이터

2. 계정 및 데이터 삭제 요청 방법

계정 및 관련 데이터 삭제를 원하시는 경우, 아래 방법 중 하나로 요청하실 수 있습니다.

  1. 이메일을 통해 요청
    다음 이메일 주소로 삭제 요청 메일을 보내주세요.
    이메일 주소: support@gitinsight.app
  2. 이메일 제목에 다음 내용을 포함해 주세요.
    [GitInsight] 계정 및 데이터 삭제 요청
  3. 이메일 본문에 아래 정보를 포함해 주세요.
    • GitHub 사용자명(GitHub username)
    • GitInsight에서 사용 중인 이메일 주소 (있을 경우)
    • 삭제를 원하는 데이터 범위 (예: 전체 계정 및 관련 데이터 삭제)

3. 처리 기간 및 보관 기간

  • 요청 접수 후 최대 30일 이내에 계정 및 관련 데이터를 삭제합니다.
  • 법적·기술적 이유로 즉시 삭제할 수 없는 데이터는, 관련 법령에서 요구하는 기간 동안만 보관 후 안전하게 삭제합니다.

4. 문의

계정 삭제와 관련하여 추가 문의 사항이 있을 경우, support@gitinsight.app 으로 연락해 주시면 됩니다.
